리눅스 명령어

cd .. 상위폴더로 이동
cd 폴더명을 하면 폴더로 이동
touch 파일 생성
ls 폴더안에 파일 보기
cp 복붙할 파일 새로만든 파일을 하면 복붙이된다

go 명령어

go mod init은 Go 언어에서 모듈(module) 초기화를 할 때 쓰는 명령어
go run  파일명.go go 파일 실행
go build 파일명.go 파일빌드를 하면 확장자가 .exe로 나오는데 그건 윈도우에서고 맥에서는 다르게나온다
